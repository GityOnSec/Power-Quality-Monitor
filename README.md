# Open-Source Power Quality Monitor (ESP32)

This project measures and logs power quality parameters including:

• Voltage (RMS)  
• Current (RMS)  
• Real Power  
• Apparent Power  
• Power Factor  
• Line Frequency  
• Voltage Sags / Swells  
• Fault Logging  

Built using ESP32 + ACS712 + ZMPT101B and a web dashboard.
